Understanding CSS Display Property

  • Block
  • Inline
  • Inline-Block
  • None

Display: Block

  • The `block` display makes an element take up the full width of its container.
  • It starts on a new line and takes up as much horizontal space as possible.
  • Common block-level elements: div, p, h1 to h6, section, etc.

Display: Inline

  • The `inline` display makes an element take up only as much width as it needs.
  • They only take up as much width as their content requires.
  • Common inline elements: <span>,<a>, <strong>, <em>, <img>, etc.
Inline Element 1 Inline Element 2 
<span class="inline-example">Inline Element 1</span>
<span class="inline-example">Inline Element 2</span>

Display: Inline-Block

  • Combines features of both block and inline.
  • Elements are laid out side by side (like inline), but you can apply block-specific properties (e.g., width, height, margins).
  • It is especially useful for creating layouts where block and inline behaviors are needed together.

Display: None

  • The element is completely removed from the document flow.
  • It does not occupy any space and is not rendered on the screen.
  • visibility: hidden:The element is not visible on the page but still takes up space in the layout.
