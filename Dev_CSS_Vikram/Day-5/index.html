IP Address

  • IP addresses are like digital addresses for devices on the internet.
  • An IP address (Internet Protocol address) is a unique string of numbers (e.g., 192.168.1.1) assigned to every device connected to a network. Think of it as a digital address that allows devices to communicate.

Server

A server is a powerful computer designed to store, process, and deliver data (like websites or apps) to other devices over a network.

  • Host websites and applications.
  • Respond to user requests sent via browsers.
  • Use various software like Apache or Nginx to manage and serve data.

DNS

The Domain Name System (DNS) acts as the internet's phonebook. It converts human-readable domain names into IP addresses, enabling browsers to locate and load the requested website.

How Websites Load in browser

When you type a URL into a browser, a series of steps occur:

  • DNS Resolution:

    • The browser contacts a DNS server to convert the domain name into its associated IP address.

  • Connecting to the Server:

    • The browser sends a request to the server using the resolved IP address.

  • Server Response:

    • The server processes the request and sends the requested HTML, CSS, JavaScript, and other resources.

  • Rendering the Page:

    • The browser parses the received files and renders the webpage for the user.
