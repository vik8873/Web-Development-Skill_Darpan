Learn CSS Selectors

Understand the different types of CSS selectors and how to use them!

  • Universal Selector
  • Type Selector
  • Class Selector
  • ID Selector
  • Attribute Selector

Universal Selector (*)

The universal selector applies styles to all elements on the page.

* { margin: 0; padding: 0; }

Type Selector

The type selector targets elements by their HTML tag name.

p { color: blue; }

Class Selector

The class selector targets elements with a specific class attribute.

.example { font-size: 18px; }

ID Selector

The ID selector applies styles to a single, unique element with a specific ID.

#unique { background-color: yellow; }

Attribute Selector

The attribute selector applies styles to elements with specific attributes.

[type="text"] { border: 1px solid gray; }

Descendant Selector

The selector .container span applies to any inside an element with the class container, no matter how deeply nested it is.

.container span {color: blue; }
