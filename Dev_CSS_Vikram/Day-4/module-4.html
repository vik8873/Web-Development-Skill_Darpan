Font Properties

font-family

The font-family property specifies the font type.

Example: This text uses font-family: 'Courier New';.

font-size

The font-size property sets the size of the font.

Example: This text uses font-size: 24px;.

font-weight

The font-weight property specifies the thickness of the text.

Example: This text uses font-weight: bold;.

font-style

The font-style property specifies whether the text is normal, italic, or oblique.

Example: This text uses font-style: italic;.

Text Properties

text-align

The text-align property aligns the text horizontally.

Example: This text is center-aligned.

text-transform

The text-transform property changes the capitalization of text.

Example: This text uses text-transform: uppercase;.

letter-spacing

The letter-spacing property adjusts the space between letters.

Example: This text uses letter-spacing: 3px;.

line-height

The line-height property sets the height of each line of text.

Example: This text uses line-height: 2;. Each line has extra spacing.
