What are JavaScript Strings?

A string in JavaScript is a sequence of characters enclosed in **single (`'`), double (`"`), or backticks (`` ` ``). Strings are used to store and manipulate text.

Strings are one of the primitive data types in JavaScript. They can contain letters, numbers, symbols, and spaces and are immutable, meaning they cannot be changed directly after creation.

Usage of Strings

            
    let str1 = 'Hello World';  // Single quotes
    let str2 = "Hello World";  // Double quotes
    let name = "John";
    let str3 = `Hello, ${name}!`;  // Backticks for template literals

10 Most Used String Methods

1. length

Returns the number of characters in a string.

console.log("Hello".length); // 5

2. toUpperCase()

Converts the string to uppercase.

console.log("hello".toUpperCase()); // "HELLO"

3. toLowerCase()

Converts the string to lowercase.

console.log("HELLO".toLowerCase()); // "hello"

4. trim()

Removes spaces from both ends.

console.log("  hello  ".trim()); // "hello"

5. slice(start, end)

Extracts part of a string.

console.log("JavaScript".slice(0, 4)); // "Java"

6. substring(start, end)

Similar to slice, but doesnâ€™t accept negative indices.

console.log("JavaScript".substring(4, 10)); // "Script"

7. replace(search, replace)

Replaces part of a string.

console.log("Hello World".replace("World", "JS")); // "Hello JS"

8. split(separator)

Converts a string into an array.

console.log("Hello World".split(" ")); // ["Hello", "World"]

9. concat()

Joins two or more strings.

console.log("Hello".concat(" ", "World")); // "Hello World"

10. includes(substring)

Checks if a string contains a substring.

console.log("JavaScript".includes("Script")); // true
