Types of Operators

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication, and division.

// Example of Arithmetic Operators in JavaScript
let a = 10, b = 5;
console.log(a + b); // Addition
console.log(a - b); // Subtraction
console.log(a * b); // Multiplication
console.log(a / b); // Division

2. Comparison Operators

Comparison operators are used to compare two values and return a boolean (true/false).

// Example of Comparison Operators in JavaScript
let a = 10, b = 5;
console.log(a > b);  // Greater than
console.log(a < b);  // Less than
console.log(a == b); // Equal to
console.log(a != b); // Not equal

3. Logical Operators

Logical operators are used to combine multiple conditions.

// Example of Logical Operators in JavaScript
let a = true, b = false;
console.log(a && b); // Logical AND
console.log(a || b); // Logical OR
console.log(!a);     // Logical NOT

4. Assignment Operators

Assignment operators are used to assign values to variables.

// Example of Assignment Operators in JavaScript
let a = 10;
a += 5; // a = a + 5
console.log(a);

a -= 3; // a = a - 3
console.log(a);

a *= 2; // a = a * 2
console.log(a);

a /= 2; // a = a / 2
console.log(a);

a %= 3; // a = a % 3
console.log(a);

a **= 2; // a = a ** 2
console.log(a);

Types of Conditional Statements

1. If Statement

The "if" statement is used to execute a block of code only if a specified condition is true.

// Example of If Statement in JavaScript
let a = 10;
if (a > 5) {
  console.log("a is greater than 5");
}

2. If-Else Statement

The "if-else" statement executes one block of code if a condition is true and another block if it is false.

// Example of If-Else Statement in JavaScript
let a = 10;
if (a > 15) {
  console.log("a is greater than 15");
} else {
  console.log("a is less than or equal to 15");
}

3. Else-If Ladder

The "else if" ladder is used to check multiple conditions.

// Example of Else-If Ladder in JavaScript
let a = 10;
if (a > 15) {
  console.log("a is greater than 15");
} else if (a == 10) {
  console.log("a is equal to 10");
} else {
  console.log("a is less than 10");
}
