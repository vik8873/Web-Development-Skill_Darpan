Understanding JavaScript Loops

Your guide to mastering loops in JavaScript

1. For Loop

The `for` loop is used to repeat a block of code a known number of times.

Example:


for (let i = 0; i < 5; i++) {
  console.log(i); // Outputs 0, 1, 2, 3, 4
}

2. While Loop

The `while` loop continues as long as a specified condition evaluates to true.

Example:


let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

3. Do-While Loop

The `do-while` loop executes the block of code once before checking the condition, ensuring that the code runs at least once.

Example:


let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);
