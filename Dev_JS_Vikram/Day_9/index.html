JavaScript Arrays Explained

Learn everything about JavaScript Arrays with examples and explanations.

What is a JavaScript Array?

JavaScript arrays are used to store multiple values in a single variable.

  • Can hold different data types.
  • Can dynamically grow or shrink.
  • Indexed starting from 0.
let fruits = ["Apple", "Banana", "Mango"];

Why Use Arrays?

  • Efficient Data Storage
  • Easy Access with Indexing
  • Powerful Built-in Methods
  • Useful for Loops and Iteration

Creating an Array

There are multiple ways to create an array:

// Using Array Literal
let numbers = [10, 20, 30, 40];

// Using new Array()
let moreNumbers = new Array(50, 60, 70);

JavaScript Array Methods

length

Returns the number of elements in an array.

let arr = [10, 20, 30];
console.log(arr.length); // Output: 3
let words = ["Hello", "World", "JavaScript"];
console.log(words.length); // Output: 3

toString()

Converts an array into a comma-separated string.

let colors = ["Red", "Green", "Blue"];
console.log(colors.toString()); // Output: Red,Green,Blue
let numbers = [1, 2, 3, 4];
console.log(numbers.toString()); // Output: 1,2,3,4

pop()

Removes the last element from an array and returns it.

let items = ["Pen", "Pencil", "Eraser"];
console.log(items.pop()); // Output: Eraser
let nums = [5, 10, 15, 20];
console.log(nums.pop()); // Output: 20

push()

Adds an element to the end of an array.

let fruits = ["Apple", "Banana"];
fruits.push("Mango");
console.log(fruits); // Output: ["Apple", "Banana", "Mango"]
let animals = ["Cat", "Dog"];
animals.push("Rabbit");
console.log(animals); // Output: ["Cat", "Dog", "Rabbit"]

slice()

Extracts part of an array without modifying the original array.

let arr = ["a", "b", "c", "d"];
console.log(arr.slice(1, 3)); // Output: ["b", "c"]
let numbers = [1, 2, 3, 4, 5];
console.log(numbers.slice(2)); // Output: [3, 4, 5]

concat()

Joins two or more arrays.

let arr1 = [1, 2];
let arr2 = [3, 4];
console.log(arr1.concat(arr2)); // Output: [1, 2, 3, 4]
let letters1 = ["A", "B"];
let letters2 = ["C", "D"];
console.log(letters1.concat(letters2)); // Output: ["A", "B", "C", "D"]

splice()

Removes or replaces elements in an array.

let arr = ["a", "b", "c", "d"];
arr.splice(1, 2, "x", "y");
console.log(arr); // Output: ["a", "x", "y", "d"]
let numbers = [10, 20, 30, 40];
numbers.splice(2, 1);
console.log(numbers); // Output: [10, 20, 40]
